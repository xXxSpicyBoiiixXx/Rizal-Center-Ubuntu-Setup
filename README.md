A simple, community-friendly script to set up old laptops with Ubuntu for the Rizal Community Center.  
The goal is to **repurpose donated or older laptops** into reliable, modern workstations with productivity, multimedia, and educational tools for the community.

---

## Quick Setup Guide

Install **Ubuntu 24.04 LTS** (recommended).  
Clone this repository:

```bash
[git clone https://github.com/xXxSpicyBoiiixXx/rizal-center-ubuntu-setup.git](https://github.com/xXxSpicyBoiiixXx/Rizal-Center-Ubuntu-Setup.git)
cd rizal-center-ubuntu-setup
```

After changing the directory please use the following commands. 

```bash
chmod +x setup.sh
sudo ./setup.sh
```
Reboot and enjoy!
